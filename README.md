 - To evaluate human factors issues through quantitative behavior data
 
## Apparatus
 - PA27 Monitor
 - Arduino Uno Rev 3 board
 - 5 direction button
 - Rotary button
 - 3D printed cover/cap
 
## Statistical Analysis
 - Two way repeated measurement ANOVA
